Crash app when launch

iPhone App

If Apple's "Calendar" is not working, the Timesheet application may not be able to start.
Launch the "Calendar" application and see if you can add your event and see it.

If you can not add a schedule, there is a possibility of a bug in iCloud.
Please see here.

Please back up contacts, calendars, reminders to prevent data loss before the trial.

Apple Watch app

Please try the following.

  1. Restart Apple Watch
  2. Uninstall and reinstall the Apple Watch app
