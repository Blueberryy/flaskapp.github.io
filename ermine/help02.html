Holiday Calendars

Holidays can be changed by setting of Holiday Calendars.
Please set the holiday calendar built into your device.
If you want to customize holidays, create a your holiday calendar and set it.
Holiday color cannot be changed.

Behavior of Holiday Calendars

  • The date of the event exists will be shown in red.
  • The event name is not shown in the month view of the app.
  • You can not create new event.

If you want to create a new event in the holiday calendar, set both the Event Calendars and the Holiday Calendars.
