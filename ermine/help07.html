How to Add a New Event

To add a new event, tap the + button on the top.

If the + button is gray, you need to set up Event Calendars first.

・How to set up Event Calendars

Select the Calendar for the Event

You can change the calendar to which you want to add events at any time when you add an event. The calendar selected at the beginning is the calendar set as "Default" in Event Calendars.
